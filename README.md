# Diamant — тестовые задания

В репозитории находятся 3 задания:

1. `email_mx_smtp_test/`  
   Проверка email через DNS MX и SMTP handshake.  
   Основной файл: `email_mx_smtp_test/email_check.py`  
   Описание: `email_mx_smtp_test/README.md`

2. `telegram_text_sender/`  
   Отправка текста из `.txt` в Telegram через бота.  
   Основной файл: `telegram_text_sender/send_text.py`  
   Тестовый текст: `telegram_text_sender/message.txt`  
   Описание: `telegram_text_sender/README.md`

3. `outreach_architecture_test/`  
   Архитектурное решение для аутрич-рассылок (multi-client, low-cost, resilient).  
   Диаграмма: `outreach_architecture_test/architecture.mmd`  
   Оценка стоимости: `outreach_architecture_test/cost_estimate.csv`  
   Описание: `outreach_architecture_test/README.md`

Дополнительно: `Блиц.md` — отдельные заметки/ответы в свободном формате.
